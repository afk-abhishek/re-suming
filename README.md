# ReSuming

ReSuming is a hybrid AI-powered resume screening system that ranks resumes
against job descriptions using classical ML and deep learning models.

## Tech Stack
- Scikit-learn (TF-IDF, SVM)
- PyTorch (BERT embeddings)
- PDF/Text Parsing
- Streamlit UI

## Features
- Resume parsing
- Job description matching
- Hybrid scoring
- Candidate ranking
- Explainable results
